# Dispatch SQLite No-Install v0.2

This version does not need `npm install`.

## New in v0.2

- Product import button
- Seed file contains 750 products from your uploaded product spreadsheet
- Product fields now include Warehouse Location and Aisle Slot
- Import updates existing product codes and adds new ones

## Start

Open Command Prompt in this folder and run:

```cmd
node server.js
```

Then open Chrome:

```text
http://localhost:3000
```

## Product Import

On the Home screen click:

```text
IMPORT PRODUCTS
```
