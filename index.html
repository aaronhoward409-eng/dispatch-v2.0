Warehouse Procedure Manual - Draft

Product import

  1. Open Home.
  2. Click IMPORT PRODUCTS.
  3. The app adds new product codes and updates descriptions, locations, and aisle slots.

Daily startup

  1. Open the Dispatch folder.
  2. Double-click or run node server.js.
  3. Open Chrome to http://localhost:3000.
  4. Backup at the end of the day.

Backups

Backups are saved in the backups folder.
